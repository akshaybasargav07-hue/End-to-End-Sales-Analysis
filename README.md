Sales Analysis Dashboard Project:
🔹Project Overview
This project analyzes sales data to generate business insights using SQL, Python, and Power BI. The goal is to support data-driven decision-making by identifying trends in sales, customer behavior, and product performance.

🔹 Tools & Technologies
Python (Pandas)
SQL Server
Power BI
Excel

🔹 Dataset
Rows: 1000
Columns: 14
Includes product, sales, customer, and transaction data
🔹 Key Analysis Performed
Total Sales by Region
Top Products by Sales
Sales by Sales Representative
Monthly Sales Trend
Profit Analysis
Payment Method Analysis

🔹 Dashboard Features
KPI Cards (Sales, Profit, Quantity)
Sales Trend (Time-based)
Region-wise Performance
Product Category Analysis
Customer Segmentation

🔹 Business Insights
High-performing regions drive most revenue
Discount impacts profit margins
Certain sales reps outperform others
Sales channel and payment method influence customer behavior

🔹 Business Recommendations
Focus on high-performing regions
Optimize discount strategies
Promote high-margin products
Improve low-performing regions
Reward top sales representatives
